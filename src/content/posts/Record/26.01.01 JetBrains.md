# CodeKey Run - JetBrains 激活工具使用指南

## 激活工具网址
`https://ckey.run/`

- CodeKey Run 是一个跨平台的 JetBrains IDE 激活工具
- 支持 Windows、Linux 和 macOS 系统
- 可自动激活 JetBrains 系列软件


# 激活方式

## Windows 激活方式
- 按下 Win + S 键，选择 Windows PowerShell (*管理员*)
- 复制命令到 PowerShell 中执行：`irm ckey.run | iex`

## Linux 激活方式
- `wget --no-check-certificate ckey.run -O ckey.run && bash ckey.run`

## MacOS 激活方式
- `curl -L -o ckey.run ckey.run && bash ckey.run`


# 注意事项
- 建议直接复制命令，避免手动输入错误
- 激活过程需要网络连接
- 工具会自动识别系统已安装的 JetBrains 产品

## MacOS特殊说明
- macOS 默认未安装 wget，建议使用 curl 命令
- 如果之前使用过其他激活工具导致激活失败，需要彻底清理
- 删除 JetBrains 相关应用的缓存文件

## 自定义激活
- 如需自定义激活信息，请访问：ckey.run
- 验证激活,激活成功后，打开任意 JetBrains IDE
- 点击菜单栏 Help → Register
